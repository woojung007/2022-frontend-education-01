# codecamp-frontend-06
코드캠프에서 공부한 내용과 자유게시판, 중고마켓 프로젝트

<br/>

## 사용한 기술 스택 
- 언어 : `Typescript` 
- 라이브러리 : `React`
- 프레임워크 : `Next.js` 
- 상태관리 : `Recoil` `Apollo-client`
- 쿼리언어 : `GraphQL` 
- 패키지 관리자 : `yarn`
- 스타일 : `Emotion`
